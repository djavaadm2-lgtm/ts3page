Cara Bergabung

Ikuti 3 langkah mudah berikut untuk mulai mengobrol bersama kami.

1

Download Client

Unduh dan instal aplikasi TeamSpeak 3 Client sesuai dengan perangkat Anda (PC/Android/iOS).

2

Masukkan Alamat

Buka TeamSpeak, klik menu Connections > Connect, lalu masukkan alamat server kami.

3

Gunakan Nickname

Gunakan format nama panggilan Anda (contoh: Budi_Jakarta) untuk memudahkan menyapa.
